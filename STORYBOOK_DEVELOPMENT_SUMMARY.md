## 技术实现细节

### 控件类型配置

1. **范围控件**: 用于数值调节（粒子数量、速度、间隔等）
2. **颜色控件**: 用于颜色选择（粒子颜色、线条颜色等）
3. **布尔控件**: 用于开关选项（动画、自动播放等）
4. **文本控件**: 用于文本输入（宽度、高度等）
5. **单选控件**: 用于枚举选择（类型、形状等）

## 运行和测试

### 启动 Storybook

```bash
pnpm storybook
```

### 访问地址

- 本地地址: http://localhost:6006/
- 网络地址: http://[your-ip]:6006/

### 构建静态版本

```bash
pnpm build-storybook
```
